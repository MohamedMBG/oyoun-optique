## 📝 Post Content

🚀 **Just launched a new full-stack luxury e-commerce & reservation platform!**

I recently finished building a complete web application for an optical boutique, focusing heavily on performance, modern aesthetics, and a seamless user experience. Over the life of this project, I architected a robust technical stack designed for scale and maintainability.

Here is a look under the hood at the architecture powering the app: 🛠️

🖥️ **Frontend & Core Framework:** 
• **Next.js 14 (App Router):** Leveraged for Server-Side Rendering (SSR) and highly optimized static generation, ensuring fast loads and great SEO.
• **React Three Fiber (Three.js):** Integrated custom 3D web graphics to render an interactive, beautiful hero scene right in the browser.
• **Tailwind CSS & Framer Motion:** Used for building a highly responsive, glassmorphism-inspired UI with smooth, premium micro-animations.

🗄️ **Backend & Database:**
• **Supabase (PostgreSQL):** Chose Supabase as the database provider for its highly scalable Postgres infrastructure and easy connection pooling mechanism. 
• **Prisma ORM:** Handled all database schema definitions, migrations, and type-safe queries, making developer experience a breeze.
• **NextAuth.js:** Implemented a secure credentials-based authentication system to protect the admin dashboard and API endpoints.

✉️ **Infrastructure & Integrations:**
• **Nodemailer:** Built a custom SMTP email integration to reliably fire off personalized reservation confirmations and status updates directly to users.
• **Vercel:** Deployed via Vercel for automated CI/CD pipelines, edge caching, and seamless Next.js optimization.

Building this platform was a fantastic experience in bridging the gap between high-end digital design (3D rendering & animations) and rock-solid backend engineering (auth, database pooling, & transactional emails). 

Always looking to learn and push the boundaries of modern web development! Check out the stack and screenshots below. 👇
